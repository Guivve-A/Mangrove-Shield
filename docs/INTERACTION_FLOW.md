# Interaction Flow - MangroveShield (Greater Guayaquil)

## Primary exploration flow

1. User opens dashboard.
2. Top status bar immediately communicates current risk state.
3. User reviews top critical zones on the left panel.
4. User clicks a zone and map flies to target.
5. Zone inspector updates quantitative metrics.
6. User adjusts storm intensity and scenario from bottom panel.
7. GSAP crossfade and KPI count-up animations reflect scenario update.
8. User switches 2D/3D view to inspect terrain and extruded risk structure.
9. User enables comparison mode to contrast two scenarios side-by-side.
10. Alerts and charts update to reflect dynamic model response.

## Analytical drill-down flow

1. Hover map feature to read contextual tooltip.
2. Click map zone to inspect local vulnerability profile.
3. Compare flood and mangrove trends in right-side charts.
4. Open model explanation to communicate methodology transparency.

## Scenario decision flow

1. Baseline scenario: Current State.
2. Toggle restoration and switch to Healthy/Restoration scenario.
3. Observe reductions in flood intensity, alerts, and risk level.
4. Switch compare target to Degraded scenario for contrast.
5. Use split map to show policy impact in demonstrations.
